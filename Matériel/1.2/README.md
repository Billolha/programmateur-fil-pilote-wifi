Version matérielle 1.2
======================

Modifications
-------------
Cette version reprend toutes les caractéristiques de la version [initiale][1] en y ajoutant les fonctionnalités suivantes :

- Ajout d'un contrôleur d'entrées sortie I2C pour libérer les I/O du Spark Core
- Possibilité de mettre deux types de relais dont un avec contact NO + NF
- Possibilité de mettre un emeteeur recepteur sans fil de type RFM12B RFM29CW ou RFM69HW
- Possibilité de mettre un recepteur 433MHz pour récupérer par exemple les trames des capteurs de temperature Orégon Scientific
- Ajout d'un connecteur I2C ET SPI pour des afficheurs OLED de type [I2C][2] ou [SPI][3]
- Ajout d'une LED (déportable ou non) sur le boitier 
- Ajout de pin out disponibles sur toutes les pattes du SparkCore
